# 发布说明

## GitHub Pages

本目录是静态主页，可以直接发布到 GitHub Pages。

推荐仓库：

- 用户主页：`你的用户名.github.io`
- 或普通作品集仓库：例如 `portfolio`

## GitHub Release 附件

`projects/03-tactical-survivor/demo/方舟幸存者可运行demo.zip` 超过 GitHub 普通仓库单文件 100MB 限制，已经写入 `.gitignore`，不要提交到仓库。

发布时需要把这个文件上传到 GitHub Release 附件。当前 Release 附件下载名为：

```text
demo.zip
```

主页和《战术幸存者》项目页会在 GitHub Pages 环境中自动生成 Release 下载链接：

```text
https://github.com/用户名/仓库名/releases/latest/download/demo.zip
```
