# 个人游戏策划作品主页

这是一个用于 GitHub Pages 发布的静态个人主页。

## 素材放置方式

- 简历 PDF：放到 `assets/docs/resume.pdf`
- 项目文档：放到对应项目目录的 `docs/` 文件夹
- 项目封面图：放到 `assets/images/`
- 小体积演示视频：放到 `assets/videos/`
- 大体积演示视频：建议使用公开视频链接，在对应项目页中替换占位链接
- 可试玩 Demo：放到对应项目目录的 `demo/` 文件夹，并保证 demo 有自己的 `index.html`

## 发布方式

将本目录上传到 GitHub 仓库后，在仓库 Settings 的 Pages 中选择从 `main` 分支根目录发布即可。
