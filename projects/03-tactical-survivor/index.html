项目概述

《战术幸存者》是一款 Unity 2D 俯视角自动攻击 Roguelite 生存 Demo。玩家选择角色后进入 10 波生存挑战，通过自动攻击、走位、拾取经验、升级四选一、波间商店和第 10 波 Boss 战形成完整局内构筑闭环。

项目首版包含开始界面、角色选择、3 名可玩角色、4 类敌人、10 波关卡、38 个升级奖励、45 个商店物品、局外成长、暂停菜单和基础反馈表现。3 名角色分别围绕金币经济、攻速暴击和重装反伤建立差异化流派，避免只通过数值高低区分角色。

我负责玩法循环、角色流派、奖励池、商店物品池、敌人与波次结构、局外成长和测试验证流程。项目使用 ScriptableObject 管理角色、敌人、波次、升级奖励和商店物品，重点展示我把玩法想法收敛成可运行 Demo、可配置数据和可展示流程的能力。

可运行 Demo 文件较大，已作为 GitHub Release 附件提供下载，附件名为“demo.zip”；项目资料包则包含策划案、截图和页面说明等轻量材料。
