# 黄忠多维建模分析结论

数据周期：2026-05-15 至 2026-06-15  
数据来源：`黄忠_横向对比其他射手_扩展版_2026-05-15至2026-06-15.xlsx`

## 为什么这样建模

单看黄忠一个英雄的数据，很容易得出“胜率健康所以不用调”或“高端出场低所以要加强”的单向结论。英雄调整不能只看一个指标，应该同时比较：

- 生态强度：胜率、出场率、禁用率。
- 分段适应性：全分段和巅峰千强的差异。
- 输出效率：分均经济、分均输出、输出占比、伤转。
- 生存容错：场均死亡、参团率、败方MVP率。
- 功能价值：对塔伤害、参团率、禁用压力。

因此本模型采用“五维评分 + 分段差异诊断 + 调整方向矩阵”。公式用于辅助排序，最终结论仍然要结合英雄机制解释。

## 黄忠诊断

黄忠属于：**大众健康但高端失配**。

这意味着黄忠不是简单的“弱英雄”。他的全分段胜率保持在 50% 以上，说明基础数值和大众分段表现并不差；但巅峰千强胜率和出场率明显走低，说明在高强度对抗中，架炮机制、站位要求和被切入风险会限制黄忠的强度兑现。

从输出效率看，黄忠的分均输出、输出占比和伤转并不低，不能把问题简单归结为“伤害不够”。从功能价值看，黄忠仍然有明显的阵地推进和对塔特色，因此这部分不应被削弱。

## 调整建议

建议方向：**机制容错优化，而不是直接加强伤害。**

可以考虑：

1. 优化架炮和收炮的操作流畅度。
2. 改善高压环境下被切入时的反制窗口。
3. 保留黄忠对塔、守塔、阵地输出的战略特色。
4. 避免直接提高基础伤害，防止全分段和低压环境强度溢出。

## 风险说明

如果直接加强伤害，黄忠在大众分段和顺风推塔体系中可能过强；如果大幅提高生存能力，又可能破坏刺客和战士对呆射的合理克制。因此更适合做小幅、可控、偏体验的机制调整。
